# we-fun

> WeFun Vue.js project

## Setting

```bash
# vue
@vue/cli 4.5.15

# node
v16.13.0

# npm
v8.1.0

# tool
VSCode

npm-cli webpack 프로젝트 생성하여 진행하였습니다.
로컬 환경에서 진행하였습니다.
초기 페이지는 http://localhost:8080/step1로 설정하였습니다.
assets/img에 캡쳐본 저장하였습니다.

감사합니다.
```

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

```
